# Vehicle Rental System

**Topic:** Python Basics  
**Complexity:** Medium  
**Estimated Time:** 6–8 hours

---

## Objectives

By completing this lab, learners should be able to:

- Apply Python data structures and functions to real-world simulation
- Design classes using inheritance and polymorphism
- Use property decorators for rentals and price management
- Implement abstract base classes for uniform rental operations
- Showcase best practices for module organization and naming conventions

---

## Description

Develop a **Vehicle Rental System** where users can rent cars, bikes, and trucks.

Each vehicle type has unique pricing and attributes. Demonstrate flexible design, showcasing OOP inheritance, properties, and polymorphism.

---

## Implementation Details

- Implement base `Vehicle` ABC and subclasses (`Car`, `Truck`, `Bike`)
- Functions to rent, return, and display availability
- Calculate rental cost dynamically based on type and duration
- Manage state (available vs rented) cleanly using control structures and collections
